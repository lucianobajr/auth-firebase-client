# Auth Firebase Client

[![Firebase CI](https://github.com/lucianobajr/auth-firebase-client/actions/workflows/ci.yml/badge.svg)](https://github.com/lucianobajr/auth-firebase-client/actions/workflows/ci.yml)


## :rocket: Technologies

- ReactJs
- Styled Components
- Lottie
- Polished
- React Toastify
- Firebase
- Yup
- Unform
- React Activity
- React Icons

## Installation

```bash
$ npm install
#or
$ yarn
```

## Running the app

```bash
$ npm run start
# development
$ yarn start
```
